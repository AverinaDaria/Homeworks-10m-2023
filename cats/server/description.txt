По url /add_cat можно добавить котика в базу данных, в теле запроса необходимо передать данные о котике: его id, имя
и цвет.

По url /find_by_name/<string:cat_name> можно получить котика по его имени, если котиков с одинаковым именем несколько -
результатом будет самый первые в хронологическом порядке. Имя котика передается в url.

По url /find_all можно получить всех котиков в базе данных

Url / не содержит содержательной информации и редиректит на /find_all
